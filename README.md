# worldcup

An **“ideal-type World Cup”** knockout tournament picker — repeatedly choose the winner of each head-to-head matchup until a single champion remains. A self-contained single-file web app (HTML + CSS + vanilla JS) with no build step.

## Highlights

- A complete single-elimination **bracket engine** in vanilla JS — pairing, advancing rounds, and crowning a winner
- Dependency-free and runs fully offline
- Clean, print-friendly visual theme

## Usage

Open `index.html` in any modern browser.
